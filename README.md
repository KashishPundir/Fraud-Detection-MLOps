# 🚨 Fraud Detection MLOps Pipeline

An end-to-end Machine Learning and MLOps project that detects fraudulent financial transactions using machine learning models and deploys them through a production-ready pipeline.

---

## 🎯 Problem Statement

Financial institutions process millions of transactions daily, making it difficult to manually identify fraudulent activities. Traditional rule-based systems often fail to detect evolving fraud patterns.

This project aims to build an intelligent fraud detection system that can:

* Detect suspicious transactions in real time
* Reduce financial losses caused by fraud
* Improve detection accuracy using machine learning
* Demonstrate industry-standard MLOps practices

### Build Image

```bash

```

### Run Container

```bash
docker run -p 8000:8000 fraud-api
```

Then run in browser: http://localhost:8000
